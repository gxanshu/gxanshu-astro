A dark, minimalist theme for the **Zed editor**, ported from Framer's code editor. This theme brings the clean visual aesthetics of Framer's code editor to Zed with a carefully matched color palette.

The theme features a dark background, bright accent blue, and vibrant syntax colors for keywords, strings, types, and other code elements — staying faithful to the original Framer editing experience.

**Installation options:**

- Install directly from Zed's extension store
- Manually place the theme file in your Zed configuration directory
- Load as a development extension

This port builds on the lineage of Vernon de Goede's original Framer Syntax theme and Adam Seckel's Framer Dark VS Code adaptation.

Check out the project on [GitHub](https://github.com/gxanshu/zed-framer-dark).
