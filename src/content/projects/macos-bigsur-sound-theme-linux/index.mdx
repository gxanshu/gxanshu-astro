A complete macOS Big Sur sound theme for Linux desktops that serves as a **drop-in replacement** for the default Ocean sound theme — every single sound event is covered, no gaps.

This project brings authentic Apple system sounds to Linux by providing all **44 sound events** from macOS Big Sur, compatible with **KDE Plasma**, **GNOME**, and **freedesktop-compliant** desktop environments.

The theme maps standard Linux sound events (login, notifications, errors, device connections) to their Big Sur equivalents. Installation is as simple as copying files to the local sounds directory and selecting the theme in system settings.

**Features:**

- All 44 freedesktop sound events covered
- Works with KDE Plasma, GNOME, and any freedesktop-compliant DE
- Can be installed from pre-built releases or compiled from source using FFmpeg
- Simple installation — just copy to `~/.local/share/sounds/` and select the theme

Check out the project on [GitHub](https://github.com/gxanshu/macos-bigsur-sound-theme-linux).
